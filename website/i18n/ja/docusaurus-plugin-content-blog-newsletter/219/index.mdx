**[Convex: Web開発者のためのリアクティブデータベース](https://convex.link/thisweek)**

Convexは、本番環境に対応したバックエンドを構築、公開、スケーリングするために必要なすべてを提供するオールインワンプラットフォームです。

TypeScriptを活用して、自動的にキャッシュされリアルタイムで動作する[クエリをコードとして記述](https://docs.convex.dev/functions/query-functions/)でき、ACID準拠の[リアクティブデータベース](https://docs.convex.dev/database)を利用できます。

そして、これだけではありません。Convexでは以下のことが可能です：

* 簡単に関数をスケジュールし、cronジョブを記述
* ファイルストレージを設定
* 効率的なサーバー関数を記述
* お気に入りのAIコーディングツールでバックエンドを生成
* その他多くの機能…

[今日すぐクイックスタートを試す](https://docs.convex.dev/quickstarts)

**[ReactアプリのUXを再考するReactile](https://www.reactile.net/?utm_source=ThisweekinReact)**

Reactileは、直感的なタイル表示とワークスペースを備えたUI/UXのターンキーフレームワークです。単一のブラウザタブ内で大規模なWebソリューション向けに並列ワークロードを促進します。Reactを使用して独自のビューコンテナ、ウィジェット、ロジックを実装し、表示はReactileに任せることができます。

さらに、Reactileは以下を提供します:

* 時間や複数デバイス間で統一されたワークフローを可能にするPromiseベースの永続化メカニズム
* タイルにネストされたタブを使用する機能
* シンプルなグローバル検索メカニズム
* CSS @containerベースのレスポンシブ対応
* その他多くの機能

バージョン2が間もなくリリースされます。30日間の無料トライアルで始めましょう。詳細はreactile.netでご確認ください。
