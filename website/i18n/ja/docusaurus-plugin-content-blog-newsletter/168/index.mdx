## ⚛️ React {#react}

[![TanStack Router 1.0](/emails/issues/168/tanstack-router.jpg)](https://tanstack.com/router/v1)

**[TanStack Router 1.0](https://tanstack.com/router/v1)**

待望のTanStack Routerが安定版1.0としてリリースされ、クリスマスに間に合う形で素敵な[🐦トレーラー](https://twitter.com/tannerlinsley/status/1738709130213560497)が公開されました。この新しいルーターは、型安全性、URL状態管理（検索パラメータ）、キャッシュなどにおいて可能な限界を押し広げています。

[SSRガイド](https://tanstack.com/router/v1/docs/guide/ssr)では`@tanstack/react-start`パッケージが言及されており、おそらく今後の[🐦TanStack Startフレームワーク](https://twitter.com/sebastienlorber/status/1742589454563369123)の予告でしょう。

見逃しているかもしれない最近のTanStackパッケージ：

* 📦 [TanStack Config - 高品質なJavaScriptパッケージを公開・維持するためのツール](https://github.com/TanStack/config)
* 📦 [TanStack Store - フレームワークに依存しない型安全なストアとリアクティブなフレームワークアダプター](https://tanstack.com/store/latest)
