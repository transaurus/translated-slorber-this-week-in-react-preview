## ⚛️ React {#react}

[![Next.js 15](/emails/issues/206/nextjs-15.png)](https://nextjs.org/blog/next-15)

[**Next.js 15**](https://nextjs.org/blog/next-15)

Next.js confに間に合うように、Next.js 15が正式リリースされました。RC2からの大きな変更はなく、いくつかの安定性の修正のみです。このv15では：

* キャッシュの動作が反転し、ほとんどのもの（例：`fetch`リクエスト）がデフォルトでキャッシュされなくなりました。
* AppルーターはReact 19 RCを使用し、ページルーターはReact 18のままです
* turbopack devの使用が推奨されるようになりました（webpackも引き続きサポートされています）
* 既存のAPIが非同期になるなどの破壊的変更があり、`@next/codemod upgrade`がアップグレードを容易にします

もちろん、DX（開発者体験）、セキュリティ、パフォーマンスの改善も数多く含まれています。
