## ⚛️ React {#react}

[![Server ActionsがServer Functionsに改名されました](/emails/issues/203/serverfunctions.jpg)](https://19.react.dev/reference/rsc/server-functions)

**[Server ActionsがServer Functionsに改名されました](https://19.react.dev/reference/rsc/server-functions)**

コード変更ではありませんが、React v19ドキュメントサイトで正式に公開された影響力のある用語変更です。Server Actionsは現在、より広範なServer Function機能のサブセットと見なされています。以下の場合にのみアクションとなります:

* `<form action>`
* `useActionState()`
* `startTransition()`

この変更により、「use server」がサーバーエンドポイントを公開するためのものであり、必ずしも変更に限定されないことが明確になりました。Server FunctionsでデータやJSXを取得することは、アンチパターンではない可能性があります。

**Next.js Server Actionsについて**: これらは`POST`リクエストを使用し、順次実行されますが、これはNext.jsの実装詳細であり、Reactはアクションを並列実行することを妨げません。Next.jsチームは、`GET`でServer Functionsを呼び出す新しいAPIを導入する意向を確認しています。
