## ⚛️ React {#react}

**[React I Love You, But You're Bringing Me Down](https://marmelab.com/blog/2022/09/20/react-i-love-you.html)**

React-Adminの作者が、Reactについて多くの側面（フォーム、useContextSelector、refs、deps配列など）で公平な批判をしています。[Dan Abramovの返答](https://twitter.com/dan_abramov/status/1572590212647391235)も見逃さないでください。

**[Next.js Layouts RFC in 5 minutes](https://vercel.com/blog/next-js-layouts-rfc-in-5-minutes)**

Next.jsのファイルシステムベースのルーターは、次のメジャーリリースで大幅にアップグレードされます。これには、Remixのようなネストされたレイアウトのサポートが含まれます。これにより、ブラウジング中にレイアウトが不必要にアンマウント/リマウントされるのを防ぐだけでなく、データの読み込みとエラーの管理も可能になります。この記事では、その概要を簡単に説明しています。[Delbaのスレッド](https://twitter.com/delba_oliveira/status/1568291558776274945)も参照してください。

**[React Router 6.4](https://remix.run/blog/react-router-v6.4)**

大きなアップデート：予想通り、多くのRemix API（データ読み込み、ミューテーションなど）がついにReact-Routerに移行されました。[2分の動画](https://www.youtube.com/watch?v=aX74DbFT5nI)をご覧ください。

**[What’s New In Gatsby 2022?](https://www.smashingmagazine.com/2022/09/new-gatsby-2022/)**

Paulは自分のブログを最新のGatsby v4にアップグレードし、最新機能の使用方法を共有しています。Deferred Static Generationを使用すると、ビルド時間を大幅に短縮できます。[Gatsby 5 Umbrella Discussion](https://github.com/gatsbyjs/gatsby/discussions/36609)と動画[Gatsby v5 Alpha Showcase](https://www.youtube.com/watch?v=C-WrnfUm33k)も参照してください：部分的なハイドレーションがやってきます！
