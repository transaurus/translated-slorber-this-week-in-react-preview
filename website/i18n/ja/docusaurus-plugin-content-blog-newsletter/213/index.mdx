## ⚛️ React {#react}

[![React 19](/emails/issues/213/react.jpg)](https://react.dev/blog/2024/12/05/react-19)

**[React 19](https://react.dev/blog/2024/12/05/react-19)**

React 19 RCはこのニュースレターで過去1年間にわたって主要なトピックの1つでしたので、正式に安定版となったこと以外に特筆すべきは、リリースを遅らせたpre-warming機能と新しいReact-DOMの静的APIの2点です。以下は既にカバーした他の機能の良い復習です：

* アクション / サーバーアクション / `useActionState()`
* `<form>` + プログレッシブエンハンスメント互換のフック
* `<Context.Provider>`の代わりに`<Context>`
* `use()`フック
* カスタム要素
* 標準プロップとしてのrefとrefコールバックのクリーンアップ関数
* メタタグの処理
* リソースのプリロード
* エラーの改善とハイドレーション差分の報告
