## 📱 React-Native {#react-native}

[![新アーキテクチャがデフォルトに、DevToolsなど新機能](/emails/issues/207/newarch.jpg)](https://reactnative.dev/blog/2024/10/23/release-0.76-new-architecture)

[**React Native 0.76 - 新アーキテクチャがデフォルトに、DevToolsなど新機能**](https://reactnative.dev/blog/2024/10/23/release-0.76-new-architecture)

React Native 0.76がリリースされ、重要なアップデートが行われました！新アーキテクチャがデフォルトで有効化され、デバッグを改善する新しいReact Native DevToolsが追加され、UI効果を強化する`boxShadow`と`filter`のCSSサポートが追加されました（新アーキテクチャでのみ利用可能）。いくつかの破壊的変更に注意してください：最低SDK要件がiOS 15.1とAndroid SDK 24（Android 7）に引き上げられています。アップグレードしてチェックしてみましょう！

v0.76は素晴らしいですが、さらに良いのは[新アーキテクチャが本番環境での使用準備完了](https://reactnative.dev/blog/2024/10/23/the-new-architecture-is-here)と公式に宣言されたことです！これは数年にわたる取り組みの成果であり、React Nativeエコシステムにとって大きなマイルストーンです！この時点で、JSI、Fabric、Turbomodules、同期型でオーバーヘッドの少ないブリッジレス通信、相互運用モードなどについてすでにご存知かもしれません。これはパフォーマンスだけでなく、ツールチップの位置決めのためにビューを同期的に計測できるようにするなど、重要なUX/DXの問題を解決するものです。私が最も興奮しているのは、おそらく最も過小評価されている機能でもあります：React NativeはついにReact 18+のコンカレント機能、特にトランジション、Suspense、そして次に来るものを利用できるようになりました。
