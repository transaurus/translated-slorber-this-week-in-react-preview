## ⚛️ React {#react}

[![TanStackDB](/emails/issues/234/tanstackdb.jpg)](https://github.com/TanStack/db)

**[TanStack DB](https://github.com/TanStack/db)**

TanStackチームからElectricSQLとのコラボレーションにより、新しいライブラリが発表されました。これは、高速なクエリエンジン、最小限の再レンダリングを実現する細かいリアクティビティ、堅牢なトランザクションプリミティブ、デフォルトでの正規化されたデータなど、多くの魅力的な機能を提供します。

このライブラリはTanStack Storeを基盤として構築されており、TanStack Queryと連携して動作します。これにより、コレクション、ライブクエリ、楽観的ミューテーションなどの機能が拡張されます。

**[Clerk Billingのご紹介](https://go.clerk.com/LphCouG)**
