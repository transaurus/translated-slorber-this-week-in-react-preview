## ⚛️ React {#react}

[![React Router 7.0](/emails/issues/211/reactrouter.jpg)](https://remix.run/blog/react-router-v7)

**[React Router 7.0](https://remix.run/blog/react-router-v7)**

計画通り、Remixの機能がReact Routerに統合され、フルスタックフレームワークの機能はViteプラグインによって実現されるようになりました。ルーティング、型安全性、ルートのプリレンダリングに関する新機能も追加されています。Remix v2ユーザーはReact Router v7へのアップグレードが推奨されており、futureフラグを有効にすればスムーズに移行できるでしょう。この大きなマイルストーンを達成したことで、チームは次の革新に集中できるようになりました。次なるMichael JacksonのRemixが楽しみですね。

* 👀 [React Core PR - moveBefore実験の追加](https://github.com/facebook/react/pull/31596): Reactチームは、[Chromiumが実装を発表した](https://bsky.app/profile/did\:plc\:hxmev3uady7j4litwnr5fzbg/post/3lbcmvgkfw227)新しいDOM API `Node.moveBefore()`をレコンシラーでサポートすることを検討中です。`insertBefore()`とは異なり、この新しいAPIは移動するDOM要素（iframeやビデオプレーヤーなど）の状態を保持できます。
