## ⚛️ React {#react}

[![Remix Vite-first](/emails/issues/178/remix_heart_vite.jpg)](https://twitter.com/markdalgleish/status/1770224011831890036)

🐦 [**RemixがVite-firstに**](https://twitter.com/markdalgleish/status/1770224011831890036)

RemixはVite統合を不安定版として導入した比較的短い移行期間を経て、これが安定版としてマークされ、すぐに新しいデフォルトとなりました。私はこの迅速な移行を見て非常に楽しみ、RemixとViteの組み合わせは強力な組み合わせだと考えています。また、ViteをRemixの開発環境およびコンパイラとして使用することで、Remixチームは表面積を減らし、コアフレームワーク機能に集中できるようになることは良いことだと思います！新しく作成されたRemixプロジェクトには、Remixプラグインを参照するVite設定ファイルしか含まれていませんが、これは簡素化の美しい例だと思います！
