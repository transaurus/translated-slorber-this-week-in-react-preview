## 💸 Sponsor {#sponsor2}

[![ReactアプリのUXを再考するReactile](/emails/issues/216/reactile.jpg)](https://www.reactile.net/?utm_source=ThisweekinReact)

**[ReactアプリのUXを再考するReactile](https://www.reactile.net/?utm_source=ThisweekinReact)**

Reactileは、すぐに使える直感的なタイル表示とワークスペースを備えたUI/UXフレームワークです。単一のブラウザタブ内で大規模なWebソリューションの並列作業を促進します。Reactを使用して独自のビューコンテナ、ウィジェット、ロジックを実装し、表示はReactileに任せられます。

さらにReactileは以下も提供します：

* 時間や複数デバイス間で統一されたワークフローを可能にするPromiseベースの永続化メカニズム
* タイル用のネストされたタブ使用の可能性
* 簡素化されたグローバル検索メカニズム
* CSS @containerベースのレスポンシブネス
* その他多くの機能

バージョン2近日リリース予定。30日間無料トライアルで始めましょう。詳細はreactile.netでご確認ください。
