## ⚛️ React {#react}

[![React 19.1](/emails/issues/228/react_19_1.jpg)](https://github.com/facebook/react/releases/tag/v19.1.0)

**[React 19.1](https://github.com/facebook/react/releases/tag/v19.1.0)**

今週、React 19.1がリリースされました。主な変更点は以下の通りです：

* 開発環境でより優れたエラー情報を提供するためにフレームワークが使用する新しい[`captureOwnerStack()`](https://react.dev/reference/react/captureOwnerStack) API。これは`children`やDOMノードの扱いにおいて既存のコンポーネントスタックとは異なります。
* Parcel 2.14の新機能と連携する、ParcelでのServer Componentsサポートの「React側」の実装。
* サーバー上でReact Server Componentsをプリレンダリングするための実験的な`unstable_prerender()`。

このリリースには、Server Componentsや`Suspense`バウンダリに関連する多数のマイナーな変更と修正も含まれています。
