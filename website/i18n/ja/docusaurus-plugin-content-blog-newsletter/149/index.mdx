## ⚛️ React {#react}

[![Panda CSS](/emails/issues/149/panda.jpg)](https://panda-css.com/)

**[Panda CSS](https://panda-css.com/)**

Chakra UIの作者であるSegun Adebayoが、待望のCSS-in-JSライブラリを公開しました（[トレーラー](https://twitter.com/thesegunadebayo/status/1669732623994834946)）。優れた開発者体験を提供し、モダンなCSS機能（カスケードレイヤー）をベースに、パフォーマンスにも優れています（ビルド時の静的解析、ランタイムなし、アトミックCSSなど）。要するに、多くの要件を満たしており、大きな注目を集めています。Reactとの統合（JSXスタイルプロップ、React Server Components互換など）に特に優れていますが、他のフレームワークでも使用可能です。

これは要注目のライブラリで、Tailwindと競合する可能性があります。ターゲットフレームワークとの統合性が高く、長い`className`の使用を強制しません。

## 📱 React-Native {#react-native}

[![RN 0.72](/emails/issues/149/rn.jpg)](https://reactnative.dev/blog/2023/06/21/0.72-metro-package-exports-symlinks)

**[React Native 0.72 - シンボリックリンクサポート、エラー改善など](https://reactnative.dev/blog/2023/06/21/0.72-metro-package-exports-symlinks)**

数回のGolden RCを経て、React-Nativeの最新バージョンがついにリリースされ、まもなくExpo 49 SDKで使用される予定です。DX、Hermes、パフォーマンスなど多くの改善点と、いくつかの破壊的変更が含まれています。特にMetroバンドラーの以下の2つの新機能が気に入っています：

* シンボリックリンク（ベータ）：モノレポやpnpmサポートを簡素化するために長らく待望されていた機能
* [パッケージエクスポート](https://reactnative.dev/blog/2023/06/21/package-exports-support)（ベータ）：ライブラリのカプセル化の向上、フロントエンドエコシステムとの互換性、特定のReact-Nativeエントリーポイントを公開する機能
