## ⚛️ React {#react}

[![Astro 3.0](/emails/issues/158/astro3.jpg)](https://astro.build/blog/astro-3/)

[**Astro 3.0**](https://astro.build/blog/astro-3/)

この新バージョンのAstroは、特に新しいView Transition APIの使用と、表示されているコンポーネントの状態を保持しながらページ間をスムーズにアニメーション遷移できる機能により革新をもたらしています。これにより多くのユースケースでSPAの必要性が減ります。

その他の注目すべき改善点：

* 画像最適化が安定版に
* レンダリングパフォーマンスの向上：Astroコンポーネントのレンダリングが30-75%高速化
* サーバーレス向けSSR機能強化：ホスティングプラットフォームへの新しい接続方法
* JSXのHMR機能強化：ReactとPreactのFast Refreshサポート
* ビルド出力の最適化：よりクリーンで高性能なHTML

**[Google Formsに代わる非クラウド型の包括的ソリューション](https://surveyjs.io/?utm_source=thisweekinreact\&utm_medium=email)**

データフローを完全に制御可能な堅牢なフォーム管理システムを構築したい場合、そしてその導入に何ヶ月も数千ドルも費やしたくない場合、SurveyJSが最適な選択肢です！

SurveyJSは4つのオープンソースJavaScriptライブラリから成るプロダクトスイートで、ドラッグ＆ドロップのフォームビルダーで動的なJSONベースの複数フォームを作成・編集し、Reactアプリでレンダリング、機密性の高い調査データを自社インフラ内に保存できます。さらにカスタム調査をブラウザ上で編集可能なPDFファイルとして表示したり、インタラクティブなチャートや表で結果を可視化できます。[詳細はこちら](https://surveyjs.io/?utm_source=thisweekinreact\&utm_medium=email)。
