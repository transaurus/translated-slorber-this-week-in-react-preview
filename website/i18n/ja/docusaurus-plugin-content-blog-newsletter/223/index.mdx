* 🛠 **完全なカスタマイズ性:** 気に入らなければ変更可能。私たちは傷つきません（*涙を拭いつつ*）。
* 💻 **優れた開発者体験:** ゴム製のアヒルを除けば、コーディングのロックスター気分を味わえます。

7年間の実績と1日100万ユーザーを誇るreact-adminは確かな選択です。[ぜひ試してみてください](https://github.com/marmelab/react-admin?ref=TWIR)。

## 📱 React-Native {#react-native}

[![State of React Native 2024 - 調査結果](/emails/issues/223/state-of-rn.jpg)](https://results.stateofreactnative.com/)

**[State of React Native 2024 - 調査結果](https://results.stateofreactnative.com/)**

調査結果が公開され、すべてのReact Nativeリソースの中で、このニュースレターが最高のポジティブスコアを獲得しました。ありがとうございます ❤️！

調査全体を要約することはできませんが、いくつかの発見を紹介します：

* React Native開発者の61%がReact（Web）から移行
* iOSとAndroidが依然として最も一般的なターゲットですが、Webも22%のシェアを占めています
* 最もポジティブな評価を得たライブラリ：Reanimated、TanStack Query、Zustand、React Navigation、Expo Router、Skia。
* スタイリングに関しては明確な勝者なし。開発者はStyleSheetやインラインスタイルなどの組み込みAPIを好んでいます。
