## ⚛️ React {#react}

[![Shadcnが新しいCLIを導入](/emails/issues/199/shadcn.jpg)](https://ui.shadcn.com/docs/changelog)

**[Shadcnが新しいCLIを導入](https://ui.shadcn.com/docs/changelog)**

大幅にパワフルで柔軟になりました：

* `shadcn add` でコンポーネント、テーマ、フック、ユーティリティ、依存関係などをインストール可能
* 複数のReactフレームワーク（Next.js、Remix、Vite、Laravel）をサポート
* Tailwind設定を上書きせずに更新可能
* コンポーネントをURLから直接インポート可能

詳細は 🐦 [このTwitterスレッド](https://x.com/shadcn/status/1829646528149143992) をご覧ください。
