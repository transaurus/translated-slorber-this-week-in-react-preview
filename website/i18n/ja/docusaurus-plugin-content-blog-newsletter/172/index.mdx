## ⚛️ React {#react}

[![Next.jsの解説](/emails/issues/172/ppr.jpg)](https://www.youtube.com/watch?v=MTcPrTIBkpA)

🎥 [**Next.js解説 - Partial Prerendering**](https://www.youtube.com/watch?v=MTcPrTIBkpA)

ビデオがここで話題になることは珍しいですが、このビデオはいくつかの理由で特別です。

まず、トピックが非常に興味深く、多くのアニメーションによる可視化とともにわかりやすく説明されています。Partial pre-renderingは[Next.js 14の実験的機能](https://nextjs.org/blog/next-14#partial-prerendering-preview)で、Next.jsのビジョンを完成させ、静的レンダリングと動的レンダリングの両方の利点を提供します。これによりNext.jsは、あらゆるレンダリングの多様性をサポートするマルチパラダイムフレームワークになります。この新機能は採用が簡単で、Suspenseの上に構築されており、新しいAPIを導入しません。フラグをオンにするだけです。

第二に、この非常にクオリティの高い動画は[Remotion](https://www.remotion.dev/)を使ってReactのWebコードで作成されています🙉。Delbaさん、素晴らしい出来栄えです！このような動画をもっと見たいですし、Remotionを使ったクリエイティブな制作プロセスについてもっと知りたいですね。関連して、Remotionチームは[GitHub Unwrapped 2023のメイキング映像](https://www.youtube.com/watch?v=WW8vIZ1kwR0)を公開しました。

## 📱 React-Native {#react-native}

このセクションは現在[Benedikt](https://twitter.com/bndkt)と共同執筆しています。コメントはメールまたは[Twitter](https://twitter.com/bndkt)でお気軽にお送りください！

[![State of React Native 2023 - 調査結果](/emails/issues/172/stateofrn.jpg)](https://results.stateofreactnative.com/)

**[State of React Native 2023 - 調査結果](https://results.stateofreactnative.com/)**

「State of ...」調査はすべて大ファンです。なぜなら、エコシステムで何が起こっているかを独自の視点で見ることができるからです。Xやその他の場所で生成された話題を追うこととは別に、開発者が実際に使用している統計を見ることはまた別のことです。そのため、昨年のState of React Nativeの結果が公開されたことを本当に嬉しく思います！そして、この調査が2年目となるため、前年との比較によってトレンドを特定することもできます。詳細については、[ぜひ結果をご自身で確認してください](https://results.stateofreactnative.com/)！私が興味深いと思った点をいくつか紹介しますが、中には本当に驚いたものもあります：

* 状態管理は非常に興味深い状況（ダジャレ意図）が続いており、Reduxはリテンション率44%（前年比57%から減少）で最下位となったものの、使用率ではReactのネイティブ機能に次ぐ78%（前年比85%から減少）で2位を維持しています。
* データ取得のカテゴリーでは、結果はソーシャルメディアで観測される注目度と一致しており、TanStack QueryとtRPCが関心度とリテンション率でトップを占め、Meta自社のRelayはMeta外では5%の使用率にとどまりほとんど無関係な存在です。
* ナビゲーションの未来はExpo Routerにあり、使用率は前年の12%から今年は40%に急増しました。
* ソーシャルメディアの活動から判断すると、スタイリングは最も話題に上るカテゴリーで選択肢が豊富です。使用率では古典的な手法（StyleSheet API、インラインスタイル、styled-components）が主流を占める中、NativeWindとTamaguiが追い上げています。Tamaguiは過去1年で最も関心を集め（UIコンポーネントライブラリとしても最高のリテンション率と関心度）、新たな競合であるreact-native-unistylesがすぐ後に続いています。
* Expoへの特別な称賛：「React Nativeツール」セクションでは、トップ10ツールのうち6つ、およびデプロイメントカテゴリーのトップ2オプションがExpo製です。
* 回答者の85%以上がReact Nativeが正しい方向に進んでいると考えており、RNアプリの構築が現在過度に複雑だと感じる開発者の割合は約30%から25%に減少しました。まだ多すぎますが、正しい方向に向かっているようです！

全体として、非常に興味深い結果であり、詳細を[チェックする](https://results.stateofreactnative.com/)価値があります！Bartłomiej氏、Kacper氏、Software Mansionの皆さんに感謝します！
