## ⚛️ React {#react}

[![State of React 2024 - 調査結果](/emails/issues/221/state-of-react.png)](https://2024.stateofreact.com)

**[State of React 2024 - 調査結果](https://2024.stateofreact.com)**

State of React 2024の結果が公開されました。情報の宝庫であり、ここですべてを語ることはできませんが、いくつかのハイライトをご紹介します：

* 「新しい」React機能の中では、Server Componentsが最も人気で、約7000人の回答者の40%が「使用したことがある」と回答（Next.jsのおかげかもしれません…）
* `forwardRef`は最も嫌われているReact APIで、廃止予定です！
* ReactのTaint APIについて聞いたことがありますか？80%のReact開発者は知らないようで、これは実験的な機能なので理解できます。
* 1位の位置に大きな変動はありません：Next.js、TanStack Query、Axios、Jest、Zodが依然として最も頻繁に使用されています。
* いくつかの新星：ZustandとVitestのユーザーが増加、TanStack Startはまだほとんど使われていませんが、関心度では1位です。

Sebが[調査の結論](https://2024.stateofreact.com/en-US/conclusion/)を書いています - 必ずチェックしてください！🙂

## 💸 Sponsor {#sponsor2}

[![Vercelコスト最適化：実戦的戦略](/emails/issues/221/pagepro.jpg)](https://pagepro.co/nextjs-hosting-cost-optimization?utm_source=newsletter\&utm_medium=email\&utm_campaign=this-week-in-react-expert-session\&utm_id=this-week-in-react)

**[Vercelコスト最適化：実践的戦略](https://pagepro.co/nextjs-hosting-cost-optimization?utm_source=newsletter\&utm_medium=email\&utm_campaign=this-week-in-react-expert-session\&utm_id=this-week-in-react)**

Next.jsアプリをVercelにデプロイするのは簡単ですが、ホスティングコストの最適化は別の話です。

PageproのChris LojniewskiとJakub Dakowiczによる**2月20日午前11時（EST）**のディープ分析セッションに参加しましょう。Vercelのコスト最適化戦略について学び、**コードレベルの最適化**（過剰な関数呼び出しを避けるためのプリフェッチ調整、適切なレンダリング方法の選択、不要なデータ転送の削除など）を通じてコスト削減する方法を理解しましょう。

さらに、**Vercelのプレミアム機能への依存を最小化**する方法（分析、画像最適化、パスワード保護を外部サービスにオフロードする「機能の外部化」）や、エンタープライズプランが価値があるケースと代替手段が適しているケースの見極め方についても解説します。**無駄な出費を止め、[今すぐ登録](https://pagepro.co/nextjs-hosting-cost-optimization?utm_source=newsletter\&utm_medium=email\&utm_campaign=this-week-in-react-expert-session\&utm_id=this-week-in-react)**してください。
