## 💸 Sponsor {#sponsor1}

[![React-admin](/emails/issues/140/reactadmin.jpg)](https://marmelab.com/react-admin/?utm_source=newsletter\&utm_term=this-week-in-react)

**[React-admin - B2Bアプリのためのオープンソースフレームワーク](https://marmelab.com/react-admin/?utm_source=newsletter\&utm_term=this-week-in-react)**

シングルページアプリケーション（SPA）は、REST/GraphQL API上に管理画面、B2Bアプリ、ERPシステムを作成するのに依然として最適な選択肢です。[React-admin](https://marmelab.com/react-admin/?utm_source=newsletter\&utm_term=this-week-in-react)は、**SPAを構築するためのリファレンスとなるオープンソースフレームワーク**です。

React-adminは150以上のビジネス向けコンポーネントを提供しています。これはUIキットではなく、React向けの最高のUIキットであるMUIを活用しています。React-adminは、開発者の生産性を向上させるために、ベスト・イン・クラスのライブラリを組み合わせ、あなたが扱うべきではない繰り返しの作業を管理します。

その最大の強みは？これまでにない最高の開発者体験で、素晴らしいドキュメントと活気あるコミュニティに支えられています。[そして、始めるのにたった30分しかかかりません](https://marmelab.com/react-admin/Tutorial.html) 🚀

React-adminは**5年間にわたって積極的にメンテナンス**されており、すでに世界中の20,000人以上の開発者に支持されています。無料で永遠に利用できますので、ぜひお試しください！😁

## ⚛️ React {#react}

[![React Labs](/emails/issues/140/reactlabs.jpg)](https://react.dev/blog/2023/03/22/react-labs-what-we-have-been-working-on-march-2023)

**[React Labs: 私たちが取り組んできたこと – 2023年3月](https://react.dev/blog/2023/03/22/react-labs-what-we-have-been-working-on-march-2023)**

Reactに関する研究開発の公式アップデートで、以下のトピックが取り上げられています：

* React Server Components。Next.jsのApp Routerを介して、すでに優れた実装が利用可能です。async/awaitを伴うRSCはクライアントアプリでも実行可能になる予定です。クライアント/サーバーサイドのモデルを統一したいという意図があり、[AsyncContext](https://twitter.com/sebastienlorber/status/1639183216799100928)がその実現に大きく貢献すると考えられます！
* アセットローディング：CSS、フォント、画像の読み込みに対するSuspenseとの統合（[詳細](https://twitter.com/acdlite/status/1637102880850059264)）。
* ドキュメントメタデータ：`<head>`メタデータに対するストリーミングサポートを備えたファーストクラスAPIの提供。
* React最適化コンパイラ：自動メモ化コンパイラとしてではなく、リアクティビティコンパイラとして紹介されています。[React Forgetは重要なマイルストーンに到達](https://twitter.com/en_JS/status/1641494974893281282)し、現在は統合作業に注力中です。
* オフスクリーンレンダリング：Reactを他のフレームワークと差別化する、私が特に期待しているイノベーションです。MetaはReact-Nativeアプリでの事前レンダリングをテストし、良好な結果を得ています。
* トランジション追跡：遅いトランジションを検出可能にします。
