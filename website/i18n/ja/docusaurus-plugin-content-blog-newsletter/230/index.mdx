## ⚛️ React {#react}

[![Next.js 15.3](/emails/issues/230/nextjs.jpg)](https://nextjs.org/blog/next-15-3)

**[Next.js 15.3](https://nextjs.org/blog/next-15-3)**

Next.jsの新しいマイナーバージョンがリリースされ、主な特徴は以下の通りです：

* Turbopack for builds（アルファ版）：8000以上のテストをパスした高速なプロダクションビルド（99%）
* Rspackのコミュニティサポート（実験的）：Webpack互換の代替バンドラー
* Client Instrumentationフック：早期のモニタリングと分析設定
* ナビゲーションフック：`onNavigate`と`useLinkStatus`でルーティングを制御
* TypeScriptプラグインの改善：大規模コードベースのサポート向上

Next.jsがTurbopackによるビルドの高速化や、Webpackの代替としてRspackを受け入れることで進化しているのは素晴らしいことです。両者には異なるトレードオフとユースケースがあります。
