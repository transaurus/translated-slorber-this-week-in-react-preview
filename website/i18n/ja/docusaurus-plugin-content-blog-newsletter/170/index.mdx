みなさん、こんにちは！

今週は素晴らしいReact関連の記事がいくつかありましたが、WakuやRemixからの興味深いリリースもありました。React Server Componentsに関しては、複雑さや所有権についてコミュニティから興味深い（ただし賛否両分の）フィードバックが寄せられました。

React Native側は落ち着いています。私たちはExpo SDK 50のリリースを心待ちにしていますが、今回はベータ版よりも大幅に遅れてのリリースとなっています。また、React Nativeがサポートする新プラットフォームであるVisionOSの正式リリースも待ち望んでいます。

私はちょうど[React経験10周年を迎えました](https://twitter.com/sebastienlorber/status/1747307299356447148)🙉 JavaScriptが大嫌いだった私のキャリアが、Reactによってこんなにも変わったとは信じがたいことです。

## ⚛️ React {#react}

[![Waku 0.19](/emails/issues/170/waku.jpg)](https://waku.gg/blog/introducing-create-pages)

**[Waku 0.19](https://waku.gg/blog/introducing-create-pages)**

このリリースでは、レイアウトとページをプログラムで作成するための新しい`createPage` APIが導入されました。これは特に、WakuがReact Server Componentsの学習体験ツールから、実際のプロダクション対応Reactフレームワークへと移行することを示しています。v0.20では軽量なファイルベースのページルートが期待されています。

**[美味しいドーナツコンポーネント](https://frontendatscale.com/blog/donuts-components/)**

この楽しく素晴らしいインタラクティブ記事は、ドーナツのアナロジー（「children」穴を持つクライアントコンポーネント）を使用して、クライアント/サーバーコンポーネントがどのように組み合わされるかを説明しています。また、より高度なRSC構成のトピックもカバーしています。今日は、その洗練されたサーバーアクションのレンダープロップのトリックについて学びました！
