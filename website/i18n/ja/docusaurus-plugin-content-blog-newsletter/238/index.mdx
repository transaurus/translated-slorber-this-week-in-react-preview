* 📜 [URL型安全性の氷山に注意](https://nuqs.47ng.com/blog/beware-the-url-type-safety-iceberg): `nuqs`ライブラリの作者が検索パラメータの潜在的な危険性について警告。美しいURLを設計し、サイズ制限内に収め、URL更新のレート制限に対処し、パラメータを検証し、古いURLを新しい検証スキーマに移行できるようにする必要があります。
* 📜 [Next.js RouteハンドラーでAsync Local Storageを使用してpropsのドリリングを防止](https://www.nico.fyi/blog/async-local-storage-to-prevent-props-drilling): React Contextのようなものですが、Node.js関数向けです。
