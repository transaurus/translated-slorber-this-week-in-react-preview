みなさん、こんにちは！

今週はニュースの面で非常にエキサイティングな週でした！

素晴らしいReactのリリースや記事がいくつかありました。React-ForgetがInstagramで徐々にロールアウトされているのを見るのは嬉しいですね。RemotionもGitHub Unwrappedのローンチでその真価を発揮し始めました。

React-Nativeは活況を呈しており、v0.73の安定版リリース、Expo SDK 50とExpo Router v3のベータ版、その他いくつかの素晴らしいリリースがありました。私が最も興奮しているのは、ExpoがユニバーサルReact Server Componentsのサポート計画を徐々に明らかにしていることです。

今週の号は私にとって特別なものです：コンテンツを一人で書いていないのはこれが初めてです。[Benedikt](https://twitter.com/bndkt)がReact-Nativeセクションのほとんどを書いてくれました。私は2020年以来React-Nativeをプロダクションで使っていないので😅、[最先端のライブラリ](https://github.com/bndkt/sharemystack)を使っている人に協力してもらい、関連性の高い洞察を提供できるのは嬉しいです。

[State of JavaScript](https://stateofjs.com/en-US)と[State of React-Native](https://stateofreactnative.com/)の調査に回答するのを忘れないでください。最後に私に言及してくれると嬉しいです！[Theo T3が方法を教えてくれます](https://youtube.com/clip/UgkxDdNASo6xNS710ODcjMx0WW4HtTxIYbrA?si=G0rAnDKOQwHqh_ED)😄

## ⚛️ React {#react}

[![Waku](/emails/issues/166/waku.jpg)](https://waku.gg/blog/introducing-waku)

[**Waku - React Server Componentの使用を強化するミニマルなReactフレームワーク**](https://waku.gg/blog/introducing-waku)

すでに紹介済みですが、v0.18の書き直し版がリリースされ、簡単な紹介ブログ記事が含まれています。
Wakuは、従来のSSG/SSR戦略から解放されたコアRSC機能を探求できるミニマルなReactフレームワークです。技術的には、純粋なJSを出力し、エッジランタイムへのデプロイをサポートするViteベースのバンドラーです。現在はVercelのスポンサーを受けています。
