## 📱 React-Native {#react-native}

[![React Native WebGPU](/emails/issues/196/rn-webgpu.jpg)](https://www.youtube.com/watch?v=i-DGY98cTjk)

**🎥 [React Native WebGPU](https://www.youtube.com/watch?v=i-DGY98cTjk)**

React Native WebGPUがリリースされ、React NativeでGPUを活用した高度なグラフィックスと計算処理が可能になりました！Web API標準を再利用することで、ThreeJSなどの既存のWebGPUコードサンプルやライブラリをReact Nativeに直接統合できます。
グラフィックスだけでなく、行列計算などにも利用可能で、オンデバイスの機械学習アルゴリズムにも有用です。Nitro Modulesを使用してTypeScriptからC++へのシームレスな相互運用性を実現。現在テクニカルプレビューとして提供されており、今後の展開が期待されます！

***

**React Nativeモジュールの未来**

React Native開発者にとってエキサイティングな時代が到来！Marc RousavyがNitroModulesで新たなアプローチを提案📦 [既存ソリューションを速度テストで凌駕](https://github.com/mrousavy/NitroBenchmarks)。VisionCameraでJSIの制約に直面した経験から生まれたこのプロジェクトは、C++側では既に安定しており、🐦 [react-native-unistyles](https://x.com/jpudysz/status/1822541758137303233)やreact-native-webgpuなど様々なプロジェクトで採用されています（SwiftとKotlinバインディングは開発中）。

👀 [一方、SwiftをTurboModulesに活用する新提案も登場](https://github.com/okwasniewski/discussions-and-proposals/blob/22edd2c7853aaf2ee654b57f4521e43d668103fb/proposals/0000-swift-turbo-modules.md)。Objective-CとC++からの移行により、ネイティブモジュール開発が大幅に改善される見込みです！
