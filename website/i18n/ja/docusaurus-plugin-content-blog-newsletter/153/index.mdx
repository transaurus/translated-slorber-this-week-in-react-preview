## ⚛️ React {#react}

[![Astro 2.9](/emails/issues/153/astro.jpg)](https://astro.build/blog/astro-290/)

[**Astro 2.9**](https://astro.build/blog/astro-290/)

Astroの新バージョンがリリースされ、待望の実験的機能である[View Transitions](https://developer.chrome.com/docs/web-platform/view-transitions/)が追加されました。これにより、クライアントサイドルーティングに頼らずに、HTMLページ間の流動的でアニメーションされたトランジションを簡単に作成できます。Next.jsやRemixのような`<Link>`コンポーネントは必要ありません。React開発者にとって、これはUXの面で従来のSPAモードと同等の魅力を持ちながら、よりシンプルなモデルを維持するフレームワークをより魅力的にします。注意点として、これらのトランジションはナビゲーション間でReactの状態を保持しないため、問題になる可能性があります。[View Transitionsのサポート](https://caniuse.com/?search=View%20Transition%20API)はまだ弱いですが、急速に進歩しており、プログレッシブエンハンスメントとして考慮できます。また、Astroはフォールバックモードを提供しています。

[**Solito 4**](https://github.com/nandorojo/solito/releases/tag/v4.0.0)
