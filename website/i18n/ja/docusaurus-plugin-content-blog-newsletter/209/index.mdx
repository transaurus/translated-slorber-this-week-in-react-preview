## 📱 React-Native {#react-native}

[![Expo 52](/emails/issues/209/expo52.jpg)](https://expo.dev/changelog/2024/11-12-sdk-52)

[**Expo SDK 52**](https://expo.dev/changelog/2024/11-12-sdk-52)

素晴らしいReact Native 0.76のリリース直後に、Expo SDK 52が登場しました。変更履歴は膨大で、多くのexpoパッケージが更新されています。特にお気に入りの項目は：

* 新規プロジェクトではNew Architectureがデフォルトで有効、既存プロジェクトではオプトイン。Expo GoはNew Architectureのみをサポート。
* Expo Router 4、React Navigation 7
* ユニバーサルサーバーコンポーネントとサーバーアクション（実験的）
* DOMコンポーネントと`’use dom’`ディレクティブによる段階的な移行またはWebViewのシームレスな統合
* React Native DevTools
* ユニバーサルツリーシェイキング（実験的）
* Reactコンパイラ（実験的）
