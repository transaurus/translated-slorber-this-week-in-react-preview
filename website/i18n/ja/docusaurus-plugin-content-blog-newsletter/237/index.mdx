## ⚛️ React {#react}

[![Storybook 9](/emails/issues/237/storybook.jpg)](https://storybook.js.org/blog/storybook-9/)

**[Storybook 9](https://storybook.js.org/blog/storybook-9/)**

Storybookは、VitestやPlaywrightとの統合により、究極のコンポーネントテストツールとなるべく進化しています。以下の機能をカバーします：

* インタラクションテスト：ユーザーの動作をシミュレートして機能を検証
* アクセシビリティテスト：WCAG違反を検出、診断、修正
* ビジュアルテスト：ピクセル単位での見た目のバグをチェック
* カバレッジレポート：テスト済みコードを一目で把握
* テストウィジェット：クリックで包括的なテストスイートを実行

このリリースには以下の機能も含まれます：

* 48%軽量化：ドキュメントとテストパッケージをオプション化してインストールサイズを削減
* ストーリー生成：ストーリーを自動的に作成するツール
* タグベースの整理：タグでストーリーをフィルタリングおよびグループ化
* ストーリーグローバル：ストーリーレベルでコンテキスト（テーマ、ビューポート、ロケール）を設定
* フレームワーク：Svelte、Next.js、React Nativeなどのメジャーアップデート！
