## ⚛️ React {#react}

[![React Server Actionsはついにフォームを修正できるのか？](/emails/issues/156/serveractions.jpg)](https://www.mux.com/blog/what-are-react-server-actions)

[**React Server Actionsはついにフォームを修正できるのか？**](https://www.mux.com/blog/what-are-react-server-actions)

Next.js 13.4（実験的機能）にすでに統合されているReactの新機能「Server Actions」についての優れた入門記事です。タイトル付きの動画アップロードの短いチュートリアルも含まれています。Server Actionsはユーザーへのフィードバックとしてレスポンスを返すことができますが、現時点では理想的な開発者体験（DX）とは言えないことが明らかです。

私の意見では、Server Actionsはプログレッシブエンハンスメントを完全にサポートすることを目指すべきです。おそらく、クライアントサイドのJavaScriptを一切使わずにフィードバックを管理するための「flash」APIの導入が必要でしょう。これは多くのバックエンドフレームワークに存在し、Remixも（[`session.flash()`](https://remix.run/docs/en/main/utils/sessions#sessionflashkey-value)）APIを提供しています。興味深い関連情報として、[`action.$$FORM_ACTION`](https://twitter.com/sebastienlorber/status/1689921046856474625)を使うとReactのプログレッシブエンハンスメントの挙動を拡張できます。
