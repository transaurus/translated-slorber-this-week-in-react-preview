## ⚛️ React {#react}

[![Next.js 15 RC 2](/emails/issues/205/nextjs.jpg)](https://nextjs.org/blog/next-15-rc2)

[**Next.js 15 RC 2**](https://nextjs.org/blog/next-15-rc2)

最初のRCから数ヶ月を経て、Next.js Confの直前に以下のような大きなアップデートが公開されました：

* `@next/codemod upgrade`: 最新のNext.jsとReactバージョンへのアップグレードを簡単に実行。
* Turbopack for development: パフォーマンス改善とNext.js 15の安定性を目標。
* Async Request APIs (Breaking): 簡素化されたレンダリングとキャッシュモデルへの漸進的移行。
* Server Actions: 推測不能なエンドポイントと未使用アクションの削除によるセキュリティ強化。
* Static Indicator: 開発中に静的ルートを表示する新しいビジュアルインジケーター。
* `next/form`: クライアントサイドナビゲーションを備えたHTMLフォームの拡張。
* `next.config.ts`: Next.js設定ファイルのTypeScriptサポート。
* `instrumentation.js` (Stable): サーバーライフサイクル可観測性のための新API。
* Development and Build improvements: ビルド時間の改善と高速なFast Refresh。
* Self-hosting: `Cache-Control`ヘッダーの制御性向上。
* ESLint 9
