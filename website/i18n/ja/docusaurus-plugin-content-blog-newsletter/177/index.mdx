## ⚛️ React {#react}

[![Pigment CSS](/emails/issues/177/pigment.jpg)](https://github.com/mui/material-ui/tree/master/packages/pigment-css-react)

[**Pigment CSS**](https://github.com/mui/material-ui/tree/master/packages/pigment-css-react)

MUIチームは、Material UI v6でEmotionを置き換える独自のゼロランタイムCSS-in-JSであるPigmentに取り組んでいます。最初のバージョンがnpmに公開され、Next.js用の`withPigment()`プラグインも含まれています。Olivierは🐦 [ツイート](https://twitter.com/olivtassinari/status/1765719897152127315)で、新しいソリューションを作成する動機と既存のソリューション（Panda、StyleX、Tailwind）との比較を説明しています。他のすべてとは異なり、これは既存のEmotionとStyled-Componentsユーザーにとっての採用を容易にします。

[![Storybook 8](/emails/issues/177/storybook.jpg)](https://storybook.js.org/blog/storybook-8/)

[**Storybook 8**](https://storybook.js.org/blog/storybook-8/)

Storybookのこの新しいメジャーバージョンには、以下のような多くの改善点があります：

* 実験的なReact Server Componentsのサポート
* Reactコントロールの自動生成の高速化（`react-docgen`経由）
* 「テストモード」での静的ビルドの高速化（CLIフラグ`--test`）
* 組み込みのビジュアルテスト（Chromatic経由）
* Viteサポートの再構築、Vitest、Vite 5
* モバイル/デスクトップのUI/UXの改善

## 💸 Sponsor {#sponsor2}

[![Refine - オープンソースのRetool代替品](/emails/issues/177/refine.jpg)](https://github.com/refinedev/refine)

**[Refine - オープンソースのRetool代替品](https://github.com/refinedev/refine)**

CRUD処理が多いアプリケーションにおいて、ローコードと従来の開発の中間的なソリューションです。

エンタープライズ向けに設計されたオープンソースのReactメタフレームワークである[Refine](https://github.com/refinedev/refine)は、管理画面、ダッシュボード、内部ツールなどのタスクに最適なヘッドレスソリューションを提供します。

ローコードツールは初期段階では有望に見えますが、スケーラビリティの問題、カスタマイズオプションの制限、ベンダーロックインのリスクなどにより、複雑なシナリオに対応できないことが多いです。

Refineは、チームが柔軟性を維持しながら、迅速に本番環境対応のアプリを開発できるようにします。

[![William Candillon - Time to Fly - React Native Skia 1.0](/emails/issues/177/skia.jpg)](https://www.youtube.com/watch?v=eUbj_NxcgKg)

🎥 [**William Candillon - Time to Fly - React Native Skia 1.0**](https://www.youtube.com/watch?v=eUbj_NxcgKg)

React Native Skiaの画期的なメジャーリリースと、インスピレーションあふれるローンチビデオが公開されました！このライブラリは2021年にWilliam CandillonとChristian Falchによって初めてリリースされ、長きにわたりShopifyから多大なサポートを受けてきました。今回ついにバージョン1.0がリリースされ、多くの改善と新機能が盛り込まれています：新しいParagraph APIにより、Skiaはリッチテキストレイアウトをレンダリングできるようになり、システムフォントとカスタムフォントを組み合わせることが可能になりました。Animations Hooksにより、既存のRN ReanimatedとGesture Handlerの統合を活用してパフォーマンスの高いアニメーションを簡単に構築できます。個人的なハイライトとして、RN Skiaはcanvaskit-jsというパッケージを介してWebプラットフォームもサポートするようになりました！
