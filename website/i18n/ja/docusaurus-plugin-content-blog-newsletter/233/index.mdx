[![Game On - React NativeにWebGPUをもたらす](/emails/issues/233/skia.jpg)](https://www.youtube.com/watch?v=t9t-VXwIc4I)

**[🎥 Game On - React NativeにWebGPUをもたらす](https://www.youtube.com/watch?v=t9t-VXwIc4I)**

37分間のビデオで、William Candillonとゲストが描画とアニメーションの未来を垣間見せてくれました。Shopifyのサポートを受けて、彼はモダンなWebGPUサポート（VulkanとMetal）をReact Nativeにもたらしています。

Web APIに準拠することで、既存のWebサンプル（Three.jsを含む）をコピー/ペーストしてすべてのプラットフォームで動作させることが可能です。また、GPUの汎用的な使用（AI、TensorFlow\.js）も可能になり、便利な低レベルTypeGPUライブラリのおかげでより簡単に実現できます。

このすべての作業は、WebGPUをベースにした新しい実験的なSkia Graphiteバックエンドを作成するために必要でした。これにより、ウェブと同様に、すべてのプラットフォームで2D/3D体験をパフォーマンスを維持しながら合成およびインターリーブできるようになり、Reanimatedとの完全な統合も実現しています。その過程で、Skiaには新機能は追加されていませんが、パフォーマンスが大幅に向上し、クラッシュが減少し、3つの新しいプラットフォーム（tvOS、macOS、Node）がサポートされました。ウィリアム、そのタコをマッサージし続けてください！
