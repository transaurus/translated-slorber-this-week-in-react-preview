## ⚛️ React {#react}

[![Meta Connect 2024でのReact](/emails/issues/204/meta.jpg)](https://engineering.fb.com/2024/10/02/android/react-at-meta-connect-2024/)

[**Meta Connect 2024でのReact**](https://engineering.fb.com/2024/10/02/android/react-at-meta-connect-2024/)

最近のイベントで、Metaは以下のことを発表しました：

* Meta Questヘッドセット向けの新しいInstagramとFacebookの複合現実アプリ。これらのアプリはReact Nativeを使用し、Static Hermes上で動作します。StyleXやReact Strict DOMなどの技術により、既存のFacebookウェブコードとインフラを再利用できました。
* メインのネイティブFacebookアプリと同等のパフォーマンスを発揮する新しいMeta Horizon RNアプリ。彼らは、render-as-you-fetchパターンが最も影響力のあるパフォーマンス改善であったと述べています。
* 最先端のReact Compiler、StyleX、React Strict DOMを使用して構築されたMeta Horizon Store（ウェブ/モバイル/HorizonOS向け）
* React Native Windows/macOS上に構築されたMeta Spatial Editorは、カスタム3Dレンダリングエンジンをネイティブに統合できます。当初はC++/Qtチームは懐疑的でしたが、最終的にはFast Refreshを気に入りました。

多くの革新的なプロジェクトがMetaで実際のプロダクションアプリに採用され、すべてのプラットフォーム向けのフレームワークとエコシステムが改善されています。これらすべてを通じて、Metaが今後数年間にわたってReactを優れたクロスプラットフォームソリューションとして継続的に投資していくことが確信できます。

🎥 [Theoによる解説動画](https://www.youtube.com/watch?v=GjBfD9PHff4)もご覧ください。

## 📱 React-Native {#react-native}

[![One - 新しいクロスプラットフォームReactフレームワーク](/emails/issues/204/one.jpg)](https://onestack.dev/)

[**One - 新しいクロスプラットフォームReactフレームワーク**](https://onestack.dev/)

TamaguiのクリエイターであるNateが、Webとモバイルをサポートする新しいユニバーサルフレームワークをリリースしました。Metroの代わりにViteをベースに構築されており、ユニバーサルな型付きファイルベースのルーティング、Remixのようなローダー、さまざまなレンダリングモード（SSG/SSR/SPA）を備えています。また、細粒度のZero同期エンジンとのオプトイン統合により、ローカルファーストアプリの構築を支援します。
