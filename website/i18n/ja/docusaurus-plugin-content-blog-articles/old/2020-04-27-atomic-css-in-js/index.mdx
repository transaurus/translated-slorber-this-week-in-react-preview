FacebookやTwitterの最近の本番環境での導入事例から、新しいトレンドが徐々に広がりつつあると考えられます：それはAtomic CSS-in-JSです。

この記事では、Atomic CSSとは何か、TailwindCSSのような機能的な/ユーティリティファーストなCSSとの関係、そして大手企業が最新のReactコードベースで採用している理由について解説します。

私はこの分野の専門家ではないため、長所と短所についての深い分析は期待しないでください。ただ、これがどのようなものなのか、概要を理解していただければと思います。

**注記**: Atomic CSSは[Atomic Design](https://atomicdesign.bradfrost.com/)とは直接の関係はありません。
