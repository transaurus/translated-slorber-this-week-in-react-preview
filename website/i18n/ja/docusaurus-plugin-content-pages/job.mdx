# This Week In React - 求人情報

こんにちは、[This Week In React](http://thisweekinreact.com/)のSebastienです 👋

このニュースレターのコンテンツ作成を手伝ってくれる「私のクローン」を探しています 😄

なぜ手伝いたいと思うでしょうか？

* 適正な報酬が支払われます
* 個人ブランドの構築に役立ちます
* 最新情報を追いながら知識を深め、その対価を得られます
* あなた自身のプロモーションのために私のオーディエンスにアクセスできます
* 他にも共通の興味があるかもしれません

理想的な候補者:

* ReactとReact-Nativeの両方に精通している
* 私が不在時に単独でニュースレターの号をキュレーション・コメントできる
* 1号あたり10-16時間の作業にコミットできる
* 長期的に継続する意思がある
* 明確にコミュニケーションを取れ、簡潔で分かりやすい言語を使える
* 興味深い弱いシグナルを察知する感度と、質の低いコンテンツを見抜く能力がある
* SNS（Twitter、LinkedIn、Instagramなど）の作法を理解している、または学ぶ意思がある

私の80%くらいのことができればおそらく大丈夫です。完全なクローンは見つからないでしょうから 😅

最初は1週間のテストから小さく始め、私が休みの時に代わりを務めてもらい、うまくいけば規模を拡大して号の半分を執筆してもらうことも可能です。
